======
Wordle
======

A stupidly feature-rich LaTeX package for typsetting Wordle puzzles.

.. code-block:: latex

    % \usepackage{wordle}
    \begin{wordle}{ANVIL}
      CRANE
      NATAL
      ANVIL
    \end{wordle}
